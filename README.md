🚚 Local Delivery System

A simple web-based local delivery system that allows customers to place orders, choose vendors, track deliveries, and view order history. Built using HTML, CSS, and JavaScript.

 Features

- 🏠 Home Page: Overview of the service  
- 🛍️ Vendors Page: Displays available local vendors  
- 🧾 Orders Page: Place new delivery or pickup orders  
- 📦 Order History: View past orders  
- 🚗 Tracking Page: Track the current delivery status  
- 🌙 Dark Mode: Toggle between light and dark themes  


 Technologies Used

- HTML5  
- CSS3 (Responsive design, Flexbox, Grid)  
- JavaScript (Local Storage for saving orders)  


 How to Run the Project

1. Clone or download the repository  
2. Open `index.html` in your browser  
3. Enjoy exploring the system!
